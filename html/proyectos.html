Experiencia Profesional

Desarrollo de Sitios Web para Empresas y Emprendimientos

He trabajado en la creación de sitios web personalizados para diversas marcas, empresas y emprendimientos, asegurando una experiencia de usuario profesional y adaptable. Utilizando tecnologías como HTML, CSS, JavaScript, y frameworks modernos, estos proyectos están diseñados para ofrecer una presencia en línea atractiva y funcional.

Software de Ventas Integrado con Siigo

Desarrollé un software de ventas en C# para PC, equipado con integración completa al sistema contable Siigo. Este sistema automatiza la recolección de datos mediante dispositivos handheld, que se sincronizan con un dashboard web para centralizar y procesar la información en tiempo real. Las bases de datos utilizadas incluyen SQLite para almacenamiento local y MySQL para sincronización en la nube.

Aplicación de Ventas para Dispositivos Android e Integrado con World Office

Desarrollé una aplicación de ventas móvil en Android que facilita la captura de datos en dispositivos como celulares y tablets. La aplicación web para la administración está diseñada con Laravel, HTML y otras tecnologías para procesamiento de datos, utilizando SQLite y MySQL para gestionar información de ventas y sincronizarla en tiempo real. Este proyecto mejora la eficiencia en campo con acceso a datos centralizados y seguros.

Aplicación de Facturación Electrónica para Estaciones de Combustible

Desarrollé una solución de facturación electrónica en Android para estaciones de combustible, que integra información de dispensadores de gasolina y genera comprobantes de pago en tiempo real. La aplicación utiliza impresoras portátiles de 58 mm y cuenta con conectividad en la nube para sincronización y almacenamiento de datos, proporcionando una solución confiable para la automatización de facturación en este sector.
